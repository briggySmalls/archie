Archie

Systems engineering without the bloat

Archie is a lightweight tool for generating model-based architecture diagrams.

Define a model...

config:
  writer: graphviz
model:
  elements:
    - name: user
      kind: actor
    - name: a
    - name: b
      children:
        - c
        - d
      associations:
        - source: c
          destination: d
  associations:
    - source: user
      destination: a
    - source: a
      destination: b/c
